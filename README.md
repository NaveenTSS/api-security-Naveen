# api-security-Naveen
Real‑world consequences of exposing an API key on GitHub
**Immediate misuse by bots**: The moment you push an API key to GitHub (even in a private repo that later becomes public)
**Financial impact**: Most APIs are Usage‑metered, Rate‑limited per key & Billable beyond free tier.
**Legal & compliance risks**: Exposed keys may be considered a security incident.

Why a company’s privacy policy prohibits logging city names:
City names are considered personal data
Location data reveals behavior
Data‑minimization principle: Collect and store only what is strictly necessary.
